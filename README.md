##### 软件说明：

1. 用途：识别屏幕下方区域中的关键图片，发现后模拟按下向下按键，跳过广告。
2. 设置：识别区域可配置，模拟按键可配置，请自行修改conf/的配置下文件。
3. 如果无法准确识别广告，请自行截取所要拦截图片，保存为bad.png。
4. 调小config中的confidence值，可提高识别模糊程度。
5. 按键：按下Esc键，暂停/恢复识别。D/Y/X/B是快捷键；\"=\"WARP 链接，\"-\"WARP 断链
